## ✨ Features

- 🎨 **Premium Dark Theme** — Cybersecurity-inspired design with neon cyan/green accents
- 📱 **Fully Responsive** — Optimized for mobile, tablet, and desktop
- ⚡ **Single File** — Pure HTML/CSS, no frameworks, instant load
- 🔤 **Terminal Aesthetic** — Monospace fonts, command-line inspired elements
- 🎭 **Animated Background** — Subtle grid pattern with glow effects
- 📊 **Stats Dashboard** — Key metrics at a glance
- 📝 **Articles Section** — Featured LinkedIn articles
- 🔗 **Social Links** — LinkedIn, GitHub, Email

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, grid, flexbox, animations
- **Google Fonts** — JetBrains Mono + Inter
